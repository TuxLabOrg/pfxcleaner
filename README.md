# pfxcleaner
### Nem sempre a Steam limpa os prefixos não usado na compatdata automaticamente, então este sript faz isso!
### As compatdatas são o WINEPREFIX usados na Steam pelo Proton, e cada jogo tem o seu e geralmente ocupa alguns gigas, mesmo quando o jogo é desinstalado não é removido.
### Esse script mostra o tamanho da pasta que sera removido e o nome do jogo a quem pertencia.
### Obs: A Steam atualiza apenas quando encerra/inicia
# BETA

Basta executar:
```
$ pfxcleaner
```
